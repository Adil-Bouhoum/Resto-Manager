# RestManager - Restaurant Management App

## Requirements
- Java 17+
- Maven 3.6+

## Installation
```bash
mvn clean install
```

## Run
```bash
mvn javafx:run
```

## Features
- ✅ Gestion menu (catégories, plats)
- ✅ Gestion tables et commandes
- ✅ Workflow cuisine temps réel
- ✅ Paiement et caisse
- ✅ Rapports

## Database
- H2 (fichier: ./data/restaurant.mv.db)
